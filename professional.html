Memory Cache Objects


Hotaru CMS has a number of variables that it loads into memory to speed up execution of

Plugins

$this->plugins = $init->plugins; $this->plugins['activeFolders'][$plugin->plugin_folder] = 1; $this->plugins['activeTypes'][$plugin->plugin_type] = 1;

Plugin Settings

$this->pluginSettings = $init->pluginSettings;

This lists the enabled plugins for the site.

The array is a key list of the plugin names with their setting values

e.g. $this->pluginSettings[$setting['plugin_folder']][$setting['plugin_setting']];
